流行方法：
+ 表格布局法
+ 行内块法

绝对定位解决方案
```css
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
```

视口单位解决方案
```css
width: 18em;
padding: 1em 1.5em;
margin: 50vh auto 0;
transform: translateY(-50%);
```

Flexbox解决方案
相对父元素
```css
display: flex;
```
居中元素
```css
margin: auto;
```
